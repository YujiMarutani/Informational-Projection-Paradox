# OAOL Brand Design Manual  
**O’Valley AI Orchestration Lab — Visual Identity System v1.0**  
**Established 2026**

---

## 1. Brand Philosophy

OAOL’s identity is built on the principle that **civilizations behave like orchestras**:  
multi‑layered, harmonic, adaptive, and sensitive to feedback.

The brand expresses:
- **Orchestration** — coordinated intelligence  
- **Civilization Dynamics** — layered systemic resonance  
- **Scientific Precision** — clarity, structure, and academic rigor  
- **Creative Harmonics** — the aesthetic lineage of O’Valley Autonomous Philharmonia

---

### 2.1 Official Names
- **Primary:** O’Valley AI Orchestration Lab  
- **Abbreviation:** OAOL

### 2.2 Tagline
**“Orchestrating Adaptive Civilization Dynamics.”**

## 3. Logo System

### 3.1 Primary Mark
A radial harmonic emblem composed of:
- Central baton (CivOS kernel)  
- Concentric feedback rings (OS layers)  
- Eight instrumental segments (system variables)  
- Polar wave field (collapse dynamics / survival engine)  
- Outer dashed boundary (civilizational envelope)

### 3.2 Variable Mapping
| Segment | System Variable | Meaning |
|--------|-----------------|---------|
| Strings | Cohesion (C) | Social binding |
| Woodwinds | Narrative (N) | Cultural meaning |
| Brass | Pressure (E) | Institutional load |
| Percussion | Criticality Phase (Φ) | Collapse threshold |
| Choir | Memory (M) | Civilizational continuity |
| Electronics | External Influence (X) | Erosion / invasion |
| Harp | Resilience (R) | Recovery capacity |
| Soloist | Governance (G) | Coordination / leadership |

### 3.3 Secondary Mark
Minimal concentric-wave emblem for:
- Academic papers  
- Dashboards  
- Monochrome contexts  

---

## 4. Color System

### 4.1 Primary Palette
| Name | HEX | Meaning |
|------|------|---------|
| Core Blue | `#0f172a` | Kernel / stability |
| Accent Green | `#22c55e` | Resilience / recovery |
| Accent Violet | `#6366f1` | Narrative / emergence |
| Accent Amber | `#eab308` | Pressure / heat |
| Accent Red | `#ef4444` | Criticality / hazard |
| Neutral Gray | `#9ca3af` | Academic tone |

### 4.2 Usage Rules
- Dark backgrounds preferred (`#050816`)  
- Avoid gradients except in wave-field animations  
- Maintain high contrast for academic readability  

---

## 5. Typography

### 5.1 Wordmark Font
**SerifOrchestraDisplay**  
- Elegant, classical  
- Reflects OAP heritage  

### 5.2 UI / Technical Font
**SansOrchestrationTech**  
- Clean, modern  
- Ideal for Streamlit apps and dashboards  

### 5.3 Monospace Layer
**MonoSystemGrid**  
- For code, formulas, and artifact labels  

---

## 6. Layout & Composition

### 6.1 Grid System
- Radial 12‑sector grid  
- Central axis alignment  
- Symmetry is mandatory  

### 6.2 Safe Zone
- Minimum margin: **15% of logo diameter**

### 6.3 Minimum Size
- Digital: **128 px**  
- Print: **25 mm**

### 6.4 Backgrounds
- Preferred: dark mode  
- Light mode requires monochrome secondary mark  

---

## 7. Application Guidelines

### 7.1 Academic Contexts
Use:
- Secondary mark  
- Monochrome palette  
- SerifOrchestraDisplay for titles  

### 7.2 Software / UI Contexts
Use:
- Primary mark  
- SansOrchestrationTech  
- Accent Green + Violet for interactive elements  

### 7.3 Posters / Slides
Use:
- Full-color primary mark  
- Radial grid layouts  
- High-contrast wave-field backgrounds  

### 7.4 Internal Documents
Use:
- Horizontal lockup  
- Neutral Gray typography
